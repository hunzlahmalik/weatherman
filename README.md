# WeatherMan
## Installation
### From Source Code
Install locally, run this in the root directory.

`pip install -e .`

## Usage
### Command Line
```bash
weatherman /path/to/weatherfiles -e 2006 -a 2012 -c 2008/05 -e 2008
```
